risk-checker

重要：
このデモは 判断・評価・診断を行いません。
危険な兆候（レッドフラッグ）を早期に可視化し、 最終判断を必ず人間に返すための設計思想を示すものです。

このページは GitHub Pages 上の公開入口です。
実装・設計・議論は GitHub リポジトリ側で管理します。

  • Core repository
    https://github.com/risk-checker/risk-checker-core
  • 設計ドキュメント： README / PRINCIPLES / ARCHITECTURE
  • Issues：議題・設計検討・ログ（2chで言う「スレ」に相当）
risk-checker is a safety-oriented design experiment.
This project intentionally avoids automated judgment.
